# 第9章 ツール利用を制御する「ポリシー」

この章で紹介するCedarポリシーのサンプルを、マネジメントコンソールのポリシー作成画面に貼り付けやすい形で掲載しています。

## ファイル一覧

| ファイル | 内容 |
|---------|------|
| `01_permit_amount.cedar` | 金額制限付きの許可ポリシー（9.2.1） |
| `02_permit_department.cedar` | 所属部署に応じたアクセス許可（9.2.4） |
| `03_forbid_role.cedar` | ユーザーの役割に基づくアクセス拒否（9.2.5） |
| `04_permit_pattern_match.cedar` | パターンマッチングによる柔軟な制御（9.2.6） |

## 9.3 【サンプルコードのみ】AgentCoreゲートウェイへのポリシー適用

各 `.cedar` ファイルの内容を、マネジメントコンソールのポリシー作成画面に貼り付けて使用します。

1. 9.3.1 マネジメントコンソールでAgentCoreの「ポリシーエンジン」を開き、ポリシーエンジンを作成する
2. 9.3.2 作成したポリシーエンジンにポリシーを追加し、`.cedar` ファイルの内容を貼り付ける
3. 9.3.3 AgentCoreゲートウェイの設定画面からポリシーエンジンを関連付ける（強制モードを選択）

> **注意**: サンプル内のARN（`arn:aws:bedrock-agentcore:us-east-1:123456789012:gateway/my-gateway`）は、ご自身の環境のゲートウェイARNに置き換えてください。
